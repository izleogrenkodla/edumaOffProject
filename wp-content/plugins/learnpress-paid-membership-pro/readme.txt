== Description ==

Paid Membership Pro add-on for LearnPress.

== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugin > Add new'.
2. Search for 'LearnPress - Paid Membership Pro'.
3. Activate LearnPress - Paid Membership Pro from your Plugins page.
