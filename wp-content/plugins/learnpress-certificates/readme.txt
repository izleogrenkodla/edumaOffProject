== Description ==  
LearnPress Certificate is an extension plugin to provide certificate for course in LearnPress.  

== Installation ==

**From your WordPress dashboard**  
1. Visit 'Plugin > Add new'.  
2. Search for 'LearnPress Certificate'.  
3. Activate LearnPress from your Plugins page.  

**From WordPress.org**  
1. Search, select and download LearnPress Certificate.  
2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
